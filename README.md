# Movie App

## Description

A movie web service built with ReactJS

### Packages

- React
- CSS
- NodeJS
- JSX
- State
- Props
- Class Components
- Data Fetching
- Routing

### Features

- Fetch movies from Torrent API
- Sort popular movies based on Torrent download counts
- Provide the movie title, poster, genre, synopsis
- Responsive User Interface
